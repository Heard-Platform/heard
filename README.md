
  # Heard

  Heard is a participatory survey platform inspired by [Polis](https://pol.is), [vTaiwan](https://vtaiwan.tw), and [Participativo Brazil](https://www.gov.br/participacoesocial/pt-br). It is a large-scale civic engagement platform, but designed for the Instagram/TikTok generation. Fast, fun, and built as a network rather than a one-off tool.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
